## 电脑入门

大多数公司用 Windows 电脑（少数是 Mac），先学会这些就够日常用了。

1. **开机 & 登录**
   - 按主机电源键（或显示器上的电源键）
   - 输入公司给的账号/密码（入职当天 HR/IT 会告诉你）
   - 桌面出来了！（如果卡住别慌，重启试试）

2. **上网**
   - 有线：插上网线自动连（大多数办公室这样）
   - Wi-Fi：右下角网络图标 → 选公司 Wi-Fi → 输入密码（问前台/行政/同事）
   - 浏览器：用 Edge 或 Chrome（公司电脑一般预装）

3. **基本设置**
   - 调亮度/音量：键盘 F 键或右下角托盘
   - 改桌面背景：右键桌面 → 个性化
   - 锁屏/注销：Win + L 快捷键（走开时记得锁屏哦）

**小贴士**：电脑卡了/蓝屏了？先重启试试，大部分问题都能解决～

## 打印机使用

打印文件是日常刚需，记住这几步基本不会出错。

1. **找打印机**
   - 问同事/看办公室地图：通常在茶水间或走廊尽头
   - 常见品牌：惠普（HP）、佳能、理光、奔图

2. **连接 & 打印**
   - Word/Excel/PDF 打开文件 → Ctrl + P（打印快捷键）
   - 选打印机（公司共享的叫“\\server\PrinterName”或直接显示型号）
   - 选份数、黑白/彩色 → 打印
   - 去打印机旁取纸（记得带工牌/饭卡，有些要刷）

3. **常见问题**
   - 打印机不显示？ → 检查电脑是否连公司网络
   - 卡纸/没墨？ → 看打印机屏幕提示，或找行政/前台报修

**小贴士**：先打一页测试页，确认没问题再打正式文件。

## 投影仪 & 会议室

开会最怕“投屏失败”，提前熟悉这些就能自信上台。

1. **有线投屏（最稳）**
   - HDMI 线（办公室常见红/蓝/黑线）
   - 一头插电脑 HDMI 口，一头插投影仪/电视 HDMI 口
   - 电脑按 Win + P → 选“扩展”或“仅第二屏幕”
   - 投影仪遥控器选对应 HDMI 输入源（HDMI1/HDMI2）

2. **无线投屏**
   - Windows：Win + K → 选会议室设备（Miracast）
   - Mac：AirPlay → 选会议室 Apple TV 或设备名
   - 有些用会议平板（如 MAXHUB、华为 IdeaHub）：直接扫码或输入码投屏

3. **会议室预订**
   - 用企业微信/飞书/Outlook 日历预订（搜索会议室名）
   - 进门刷卡/扫码开门

**小贴士**：提前 5 分钟到会议室测试投屏，免得开会尴尬。

## 采购申请流程

想买鼠标、键盘、U盘、纸笔？走这个流程。

1. **确认需求**
   - 先问组长/直属领导：是否真的需要，公司是否有库存

2. **填写申请**
   - 用企业微信/飞书/钉钉/公司 OA 系统 → 搜索“采购申请”或“物品领用”
   - 填：物品名称、型号、数量、预计金额、用途（写清楚“用于日常办公，提高效率”）
   - 附上截图/链接（淘宝/京东链接最好）

3. **审批流程**
   - 提交 → 领导审批（组长 → 部门经理 → 行政/财务）
   - 审批通过 → 行政统一采购，或你自己买后凭发票报销

4. **到货后**
   - 领用/签收 → 放自己工位或公共区

**小贴士**：小额（<200元）的东西很多公司允许先买后报销，先问清楚政策。

## 小Tips & 常见问题速查

- 电脑没声音？ → 右下角喇叭图标 → 点开选正确输出设备
- 投影没画面？ → 检查 HDMI 线是否插紧 + Win + P 切换
- 打印机没反应？ → 重启打印机/电脑，或问行政
- 想买东西不知道批不批？ → 先问领导/行政，写明“办公必需”
- 忘记密码/账号？ → 找 HR/IT 重置（别乱猜哦）
