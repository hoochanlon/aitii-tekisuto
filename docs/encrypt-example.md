# 文章内容加密使用示例

本文档演示如何在 MkDocs 文章中使用内容加密功能。

## 使用方法

在 Markdown 文件的 front-matter 中添加 `password` 字段，整个页面内容会自动加密：

```markdown
---
password: your-password
---

这里是要加密的内容。

可以包含多行文本、**Markdown 格式**、代码块等。

```python
def example():
    print("加密的代码示例")
```

更多内容...
```

## 说明

1. 在 front-matter 中添加 `password` 字段指定密码
2. 整个页面内容会自动加密，无需额外的加密标记
3. 访问页面时，需要输入密码才能查看内容

**优势**：
- ✅ 密码不会出现在正文中，更安全
- ✅ 符合 MkDocs 的标准做法
- ✅ 使用简单，只需在 front-matter 添加密码即可
- ✅ 便于管理和维护

## 特性

- ✅ 使用 AES-CBC 256 位加密
- ✅ 客户端解密，无需服务器端处理
- ✅ 密码自动保存（24小时有效）
- ✅ 支持 MkDocs Material 主题样式
- ✅ 支持暗色模式

## 示例

下面是一个加密内容的示例：

---
password: demo123
---

这是一段加密的内容。只有输入正确的密码（demo123）才能查看。

**注意**：密码区分大小写。

## 注意事项

1. 密码应该足够复杂，以确保安全性
2. 加密内容在构建时加密，运行时解密
3. 密码会保存在浏览器的 localStorage 中（24小时有效）
4. 每次访问页面都需要输入密码（除非已保存）
