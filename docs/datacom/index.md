# 数据通信 (Datacom)

本模块聚焦 **数据通信网络** 的核心技术、架构设计、设备配置、故障排查与优化实践，旨在帮助网络工程师快速构建高效、稳定、安全的企业级 / 园区级 / 数据中心网络环境。

与 **AD 域控** 模块配合，形成“身份认证 + 网络通信”的完整企业 IT 基础设施知识体系。

---

## 模块整体脉络

数据通信部分围绕一条主线展开：

1. **电话网络**：从传统语音到 IP 语音，理解话机、分机、语音网关、程控交换等如何接入与编排。
2. **电脑网络**：从 OA 办公网入手，逐步构建对 IP 地址、子网掩码、网关、VLAN、路由等基础概念的体系化认知。
3. **设备网络**：以主备堆叠 + 核心 / 汇聚 / 接入三层架构为主线，讲清典型企业园区网、机房网络的设计与实施方法。
4. **配置备份**：覆盖交换机、防火墙等关键设备的备份策略与实战操作，确保网络出现故障时可以“有据可还原”。
5. **故障排除**：结合真实事故案例，给出从“现象 → 定位 → 根因 → 复盘”的完整排障思路与 SOP。

读完本模块，你应该能做到：

- 能看懂一张典型的企业网络拓扑（核心堆叠 + 汇聚 + 接入 + 出口 + 电话系统）；
- 能说清一台 PC / 电话从接入到出网的关键路径与涉及设备；
- 碰到常见网络异常时，不再只会“重启设备”，而是能按步骤有条理地排查。

建议的阅读顺序（可根据实际工作重心调整）：

1. 电脑网络 → 打好 IP / VLAN / 路由 / 网关 等基础；
2. 设备网络 → 结合主备堆叠三层架构，形成整体网络设计观；
3. 电话网络 → 将语音业务纳入统一网络架构视角；
4. 配置备份 & 故障排除 → 用真实案例和运维实践加深记忆与肌肉记忆。
