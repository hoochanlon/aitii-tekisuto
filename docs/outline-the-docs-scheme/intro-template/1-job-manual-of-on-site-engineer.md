## 一、项目概述

**甲方公司简介** 甲方公司成立于1996年，是国内知名的金融贸易服务企业，在全国交易市场体系中占据重要地位。公司主要为企业及个人提供各类零售金融产品与服务，同时开展资产管理、社保基金管理、城市便民服务等多元化业务。

**乙方公司服务范围** 本项目由乙方公司承接，主要负责甲方公司驻点现场的**IT运维支持**，核心工作内容包括：

- 主机（PC）软硬件日常维护与故障处理
- 打印机、扫描仪等常用办公外设的维保
- 网络接入、故障排查及基础信息安全保障

## 三、岗位职责与要求

### 3.1 主要职责

1. 保障终端用户桌面环境的稳定，快速响应并解决软硬件问题
2. 维护公司网络的正常运行，协助做好基础信息安全工作
3. 负责IT资产的登记、使用管理及合理利用

### 3.2 任职要求

- 熟悉PC硬件结构及常见桌面操作系统故障判断
- 熟练掌握打印机、扫描仪等办公外设的日常使用与常见故障处理
- 具备计算机网络基础，能独立处理90%以上的常见网络问题
- 良好的服务意识、沟通表达能力和客户导向思维

## 四、工作流程与服务方案

### 4.1 日常工作闭环流程

1. 接到用户报修（电话 / 企业微信 / 工单）→ 5分钟内响应
2. 前往现场（或远程指导）→ 诊断 & 解决
3. 处理完毕 → 向报修人确认效果 → 登记《每周维护记录表》
4. 每周五下班前将本周记录汇总发送至企业微信群（参考模板）

**特别说明**

- 主机硬件故障 → 联系原厂或代理商维保
- 打印机/扫描仪硬件故障 → 先报备甲方对接人，再联系厂商
- 涉及采购/新设备入网 → 走甲方OA签报 → 批准后由客户经理跟进采购

## 五、网络维护要点

1. **上网前必查**：认证客户端、杀毒软件 / 终端管控软件是否正常运行
2. **换位置 / 换主机** → 通知甲方科技部刷新账号绑定
3. **内外网切换** → 填写《网络变更申请单》→ 由甲方分配新IP
4. **端口不通** → 检查网线是否已连到楼层交换机对应端口
5. **VPN / 代理设置** → 使用甲方提供的账号、地址、端口

**网络环境概况**

- 内网带宽 ≈ 10 Mbps
- 外网带宽 ≈ 1 Mbps（受QoS策略控制）
- 外网访问需通过认证客户端授权
- 企业邮箱域：@xxx.com

## 六、注意事项（必读）

1. 节假日保持电话24小时畅通，使用礼貌用语确认部门及姓名
2. 涉及硬件升级、更换需提前向甲方书面反馈并获批
3. 系统重装前**必须**与用户再三确认资料已备份
4. 遇到无法独立解决的问题，第一时间联系**甲方科技部**或**乙方项目经理**

## 七、乙方公司内部管理要求

**远程管理**

- 每日企业微信打卡（考勤、请假、加班）
- 每月参加公司例会一次
- 每月提交一篇驻点工作总结（进度 + 问题 + 优化建议）

**请假流程** 甲方使用部门同意 → 乙方组长同意 → 企业微信提交请假申请 → 审批通过

**离职交接** 提前1个月提交申请 → 完成业务/文档/账号交接 → 新人得到甲方认可 → 签署《离职交接表》→ 正式办理离职
